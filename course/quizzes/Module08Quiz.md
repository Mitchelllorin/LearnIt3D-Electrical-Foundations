## Module 8 Quiz — Transformers & Induction

**Instructions:** Answer all 10 questions. Show working for transformer-ratio calculations.

## Questions

1. **Multiple Choice**  
   A basic transformer depends on:
   - A. Steady DC only
   - B. Changing magnetic flux
   - C. Chemical reaction in the core
   - D. Neutral current only

2. **Short Calculation**  
   A transformer has `Vp = 2400 V`, `Np = 200`, and `Ns = 20`. Calculate the secondary voltage `Vs`.

3. **Fill in the Blank**  
   A transformer is called **step-down** when the secondary has **__________** turns than the primary.

4. **Short Calculation**  
   A control transformer has `Vp = 120 V`, `Vs = 24 V`, and `Np = 500` turns. Calculate the number of secondary turns `Ns`.

5. **True / False**  
   An autotransformer provides the same isolation as a two-winding isolation transformer.

6. **Multiple Choice**  
   Laminated transformer cores are mainly used to reduce:
   - A. Neutral current
   - B. Eddy current losses
   - C. Frequency
   - D. Copper colour changes

7. **Short Calculation**  
   An ideal transformer steps **25 000 V** down to **240 V**. If the primary current is **2 A**, what is the approximate secondary current?

8. **Fill in the Blank**  
   Transformer capacity is commonly rated in **VA** or **__________**.

9. **Multiple Choice**  
   A centre-tapped residential transformer secondary commonly provides:
   - A. 24 V DC only
   - B. 120/240 V split-phase
   - C. 347/600 V three-phase only
   - D. 208 V single-phase only

10. **True / False**  
    Applying steady DC to a standard transformer can cause excessive primary current and overheating.

## Answer Key

1. **B** — Transformers require changing flux for induction.  
2. **240 V** — `Vp / Vs = Np / Ns = 200 / 20 = 10`, so `Vs = 2400 / 10`.  
3. **Fewer**  
4. **100 turns** — `Ns = Vs × Np / Vp = 24 × 500 / 120`.  
5. **False** — An autotransformer does not provide the same isolation.  
6. **B** — Laminations reduce eddy currents in the core.  
7. **About 208.3 A** — `Is / Ip = Vp / Vs = 25000 / 240`, so `Is ≈ 2 × 104.17`.  
8. **kVA**  
9. **B**  
10. **True**

# LearnIt3D — Electrical Foundations

Electrical Foundations Apprenticeship Prep course under the LearnIt3D brand.

## What is included

- Full original course files in this repository
- Installable web app support (desktop icon + standalone launch)
- Course cover image for install and sharing metadata
- Direct download package link: https://github.com/Mitchelllorin/LearnIt3D-Electrical-Foundations/archive/refs/heads/main.zip

## 1-click install flow

1. Open the hosted `index.html` entry point in a browser that supports app install.
2. Click **Install App** from the in-app install strip.
3. Launch the course from the created desktop icon (no manual `index.html` browsing needed afterward).

## Offline support

The app now ships with a manifest and service worker so installed users can open the course from their desktop icon and continue to access previously cached course content offline.

# LearnIt3D — Electrical Foundations Apprentice Prep

This repository contains the full course content and offline-ready 3D interactives.

## Quick download

1. Click **Code** on this repository page.
2. Click **Download ZIP**.
3. Extract the ZIP file.

## Easy run (recommended)

If you prefer running from a local server:

- **Windows:** double-click `launch-course.bat`
- **Mac/Linux:** run `bash launch-course.sh`

Then open:

`http://127.0.0.1:4173/course/3d/index.html`

## Notes

- 3D scenes now use local Three.js files in `/course/3d/vendor`.
- No CDN access is required to run the course interactives.
- A ready-to-download package is included as `LearnIt3D-Electrical-Foundations-Offline.zip`.
